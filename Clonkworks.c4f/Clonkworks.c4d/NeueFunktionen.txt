  int   GetViewRange(object pObj)
  int   GetFoW(int iPlayer, int iFunc)
  any   GetGraphics(int iSay, object pObj, int iOverlay)
  int   GetOverlayByIndex(int iIndex, object pObj)
  int   GetLocalCount(object pObj, int iFunc)
  any   GetLocalByIndex(int iIndex, object pObj, int iFunc)
  bool  CheckLocal(string szName, object pObj)
  int   FindCommand(string szName, object pObj, int iStartAt)
  int   GetSpeed(object pObj, int iPrecision)
  bool  ResetCon(object pObj)
  bool  MorphTo(id idMorphTo, object pObj)
  bool  MorphBack(object pObj)


  array Find_Command(int iElement, any iEqual)
  array Find_ColorDw(int iColor)
  array Find_Effect(string iName, int iPrio)
  array Find_Local(any iLocal, any iEqual)
  array Find_FuncEqual(string iFunc, any iEqual)
  array Find_Team(int iTeam)
  array Sort_XDistance(int iX, int iDir)




GetViewRange:
 Gibt die Sichtweite eines Objekts zurück.

 int GetViewRange(object pObj)
  pObj: Objekt, dessen Sichtweite abgefragt werden soll.

 Hinweis: Diese Funktion ist zwar schon in "GetXVal.c" als "GetObjPlrViewRange" vorgegeben, doch für mich ist der Name etwas zu lang (unpassend) als dass man auf ihn kommen könnte.


GetFoW:
 Gibt zurück, ob der jeweilige Spieler beschränkte Sichtweite hat.

 int GetFoW(int iPlayer, int iFunc)
  iPlayer: Spieler, dessen FoW abgefragt werden soll.
  iFunc: Wenn WAHR, wird der Nebel, wenn durch Maus erschaffen, auch mitgezählt.

 Vorsicht: Laut "GetXVal.c" kann diese Funktion zu Synchronisationsfehlern führen.


GetGraphics:
 Fragt Graphikdaten ab. Der Rückgabetyp richtet sich nach dem abgefragten Wert. Wurde die Graphik nicht gefunden, wird immer 0 zurückgegeben.

 any GetGraphics(int iSay, object pObj, int iOverlay)
  iSay: Index des Wertes, der abgefragt werden soll.
   0 = idSrcDef		(Die ID aus der die Graphik stamm)
   1 = szGfxName	(Der Name der Graphik)
   2 = iOverlayMode	(Die Angabe, wie das Overlay gezeichnet ist)
   3 = szAction		(Nur für iOverlayMode==GFXOV_MODE_Action: Die Aktivität in der Definition idSrcDef, dessen Grafik über das Zielobjekt gelegt ist)
   4 = dwBlitMode	(Der Blitmodus der überlegten Grafik)
   5 = pOvelayObject	(Nur für iOverlayMode==GFXOV_MODE_Object: Das Objekt, dessen Grafik über das Zielobjekt gelegt ist)
  pObj: Objekt, dessen Graphik abgefragt werden soll.
  iOverlay: Index der abzufragenden Overlaygrafik. Bei 0 wird die normale Graphik des Objektes abgefragt


GetOverlayByIndex:
 Gibt die indizierte Graphiknummer zurück.

 int GetOverlayByIndex(int iIndex, object pObj)
  iIndex: Index in der Overlayliste.
  pObj: Objekt, dessen Graphik abgefragt werden soll.


GetLocalCount:
 Liefert die Anzahl der verfügbaren Locals im Objekt.

 int GetLocalCount(object pObj, int iFunc)
  pObj: Objekt, dessen Localanzahl abgefragt werden soll.
  iFunc: Art der Locals, die abgefragt werden sollen.
   0 = Alle benannten Locals				(LocalN("..."))
   1 = Alle unbenannten Locals mit irgendeinem Wert	(Local(...))


GetLocalByIndex:
 Gibt den Namen/die Nummer des Locals im Objekt zurück.

 any GetLocalByIndex(int iIndex, object pObj, int iFunc)
  iIndex: Index in der Localliste.
  pObj: Objekt, dessen Local abgefragt werden soll.
  iFunc: Art der Locals, die abgefragt werden sollen.
   0 = Alle benannten Locals				(LocalN("..."))
   1 = Alle unbenannten Locals mit irgendeinem Wert	(Local(...))


CheckLocal:
 Prüft, ob sich das genannte Local im Objekt befindet.

 bool CheckLocal(string szName, object pObj)
  szName: Name des Locals.
  pObj: Objekt, das nach dem Local geprüft werden soll.


FindCommand:
 Gibt die nächste, um eins erhöhte Nummer von einem Command mit demselben Namen wie szName zurück. Sollte keines mehr gefunden werden, so gibt es 0 zurück.

 int FindCommand(string szName, object pObj, int iStartAt)
  szName: Der Name des Commands, nach dem gesucht wird.
  pObj: Das Objekt in dem gesucht wird.
  iStartAt: Wenn angegeben, so wird ab dieser Nummer der Suchvorgang gestartet


GetSpeed:
 Liefert die Geschwindigkeit von pObj.

 int GetSpeed(object pObj, int iPrecision)
  pObj: Objekt dessen Geschwindigkeit zurück gegeben werden soll. Null bei lokalem Aufruf.
  iPrecision: Genauigkeit mit der die Geschwindigkeit zurückgegeben werden soll. (Standardwert: 10)


ResetCon:
 Setzt die Fertigstellung/Wachstum des Objektes auf genau 100,000.
 Dabei werden keine Funktionen wie "Initialize" im Objekt aufgerufen.

 bool ResetCon(object pObj)
  pObj: Zielobjekt (bei lokalen Aufrufen Null)


MorphTo:
 Verwandelt den Clonk in einen anderen Clonk.
 Beim Zurückverwandeln werden alle Locals, Graphiken usw. im Clonk beibehalten.
 Es werden dabei keine Funktionen wie "Redefine" im Clonk ausgelößt.

 bool MorphTo(id idMorphTo, object pObj)
  idMorphTo: Die ID des Clonks, in den das Objekt verwandelt werden soll. Bei 0 wird der Clonk zurückverwandelt.
  pObj: Objekt, das verwandelt werden soll (bei lokalen Aufrufen Null).


MorphBack:
 Setzt jegliche Verwandlung des Clonks durch "MorphTo" zurück.

 bool MorphBack(object pObj)
  pObj: Objekt, das zurückverwandelt werden soll (bei lokalen Aufrufen Null).




Find_Command:
 Suchkriterium: Sucht alle Objekte mit dem entsprechenden Kommando.

 array Find_Command(int iElement, any iEqual)
  iElement: Information, die zurückgegeben werden soll (Siehe SetCommand()).
   0 = szCommand
   1 = pTarget
   2 = iX
   3 = iY
   4 = pTarget2
   5 = iData
  iEqual: Zu vergleichendes Wert.


Find_ColorDw:
 Suchkriterium: Sucht alle Objekte mit der entsprechenden Besitzerfarbe.

 array Find_ColorDw(int iColor)
  iColor: Farbe, die gesucht wird.


Find_Effect:
 Suchkriterium: Sucht alle Objekte mit dem entsprechenden Effekt.

 array Find_Effect(string iName, int iPrio)
  iName: Suchmaske für den Namen der Effekte. Es können '*' und '?' in der Maske verwendet, oder auch direkt Effektnamen angegeben werden. Bei 0 werden alle Effekte gezählt.
  iPrio: Wenn ungleich 0 werden nur Effekte zurückgegeben, die höchstens die angegebene Priorität haben.


Find_Local:
 Suchkriterium: Sucht alle Objekte mit dem entsprechenden Local.

 array Find_Local(any iLocal, any iEqual)
  iLocal: Name/Nummer des Locals, das gesucht wird.
  iEqual: Wert, welches das Local betragen soll.


Find_FuncEqual:
 Suchkriterium: Sucht alle Objekte bei denen die vorgegebene Funktion gleich iEqual ist.
 
 array Find_FuncEqual(string iFunc, any iEqual)
  iFunc: Funktion, dessen Wert verglichen wird.
   Vorsicht! Hier muss man im Gegensatz zu "Find_Func" die Funktionsklammern am Ende der Funktion hinzufügen.
  iEqual: Wert, welches die Funkion herrausgeben soll.


Find_Team:
 Suchkriterium: Sucht alle Objekte dessen Besitzer im Team iTeam ist.

 array Find_Team(int iTeam)
  iTeam: Das Team, dem das Objekt gehören soll.


Sort_XDistance:
 Sortierkriterium: Sortiert alle gefundenen Objekte nach deren X-Koordinaten.

 array Sort_XDistance(int iX, int iDir)
  iX: X-Koordinate, von der aus sortiert wird.
  iDir: Richtung, zu der die Objekte geordnet werden.
   0 = Links
   1 = Rechts
   2 = Beide
